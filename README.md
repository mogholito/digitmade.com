# DigitMade Website

A single page website for DigitMade.

## Features

- Responsive design
- Font Awesome icons
- jQuery integration
- Minimal and clean structure

## Structure

```
├── index.html                    # Main page
├── cgv-hebergement.htm          # Terms & conditions page
├── assets/                       # Minified assets
│   ├── script.min.js
│   └── style.css
├── css/                         # Stylesheets
│   ├── font-awesome.min.css
│   └── main.css
├── js/                          # JavaScript files
│   ├── scripts.js
│   └── libs/                    # Third-party libraries
│       ├── jquery.min.js
│       └── jquery.tweet.js
├── img/                         # Images
└── infogestion/                 # Info & Gestion section
    ├── index.html
    ├── css/styles.css
    └── js/script.js
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
