# Project 260600 - BENLYSTA Email Template

## Description
Email template for healthcare professionals about BENLYSTA® (belimumab), a treatment for systemic lupus erythematosus (SLE).

## Client
GSK (GlaxoSmithKline)

## Content
Medical information email featuring:
- Patient case study (Madame Dupont)
- Diagnostic criteria for systemic lupus
- Treatment indications and prescribing information
- Product information for BENLYSTA® IV and SC forms

## Files
- `index.mjml` - MJML source template
- `index.html` - Compiled HTML email
- `img/` - Image assets

## Technologies
- MJML (responsive email framework)
- HTML/CSS
